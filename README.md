📊 Sales Forecasting & Time Series Analysis (Power BI Project)
📌 Project Overview

This project focuses on performing end-to-end data analysis using Power BI, starting from raw CSV data to generating actionable insights. It includes data cleaning, transformation, time series analysis, and interactive dashboard creation, all within Power BI.

The goal is to analyze historical sales data, identify trends, and enable data-driven decision-making through powerful visualizations.

🎯 Objectives
Perform data cleaning directly in Power BI
Analyze sales trends using time series techniques
Create calculated columns and tables using DAX
Build interactive dashboards for business insights
Support strategic decision-making

🛠️ Tools & Technologies Used
Power BI
Power Query (Data Cleaning & Transformation)
DAX (Calculated Columns, Measures, and Tables)
Data Modeling
Interactive Dashboard & Visualizations
CSV Dataset (Data Source)

📊 Project Features
📈 Time Series Analysis (trend & seasonality insights)
📊 Sales Forecasting using Power BI analytics
🧮 DAX Calculations (KPIs, Profit Margin, Measures)
🖥️ Interactive Dashboard (filters, slicers)
📉 KPI Tracking (Sales, Profit, Profit Margin)

⚙️ Steps Performed
Imported CSV data into Power BI
Cleaned and transformed data using Power Query
Created calculated columns and tables using DAX
Built data model and relationships
Performed time-based analysis (yearly trends)
Designed interactive dashboards with visuals 
Generated insights for business understanding
